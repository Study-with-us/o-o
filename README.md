# 스터디 진행 방식

- 매주 관심있어 하는 공부 주제를 선정
- 해당 내용을 학습하고 설명할 발표 자료를 작성
- 해당 내용을 발표하고 같이 리뷰한다
- 발표 자료는 Repo에 Push하여 공유한다
    - 발표에 사용된 소스코드는 개인 Repo에 작성한 뒤, Repo 주소를 첨부한다
- 발표 후 스터디 진행 후 해산
